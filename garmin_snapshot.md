# Garmin Snapshot Tracker (Weekly)

Use this file to log a quick Sunday snapshot from Garmin Connect.

## How to use

Every Sunday, add one entry with:
- Resting HR (4w avg + current)
- HRV status + 7-day avg
- Sleep score (4w avg)
- Training Readiness
- Acute Load / Load Focus
- VO2 Max (if shown)
- Notes (how legs felt)
